Algunos lenguajes de la web

HTML+CSS+Javascript

  • HTML 5

    • HTML5 (Hyper Text Markup Languaje, versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML

    Usa la extensión.html

  • CSS 3

    • Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets) es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito HTML.

    Usa la extensión.CSS

  • JavaScript

    • JavaScript (abreviado comúmmente "JS") es un lenguaje de programación interpretado utilizado principalmente en el lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámica.

    Usa la extensión.js
